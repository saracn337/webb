4. Creamos gráficos

Gráficos

Una vez que tenemos los datos organizados en una tabla, podemos representarlos con gráficos. Los gráficos nos ayudan a ver la información de forma más visual y a entenderla mejor.

Los dos tipos más comunes que vamos a usar son:

  • Gráfico de barras
    • Sirve para comparar cantidades.
    • Cada barra representa una actividad y su altura indica cuánto tiempo se dedica a ella.
    • Es muy útil para ver rápidamente qué actividad ocupa más o menos tiempo.
  • Gráfico circular (o de sectores)
    • Sirve para ver cómo se reparte un total.
    • El círculo se divide en partes, y cada parte representa una actividad.
    • Cuanto más grande es la parte, más tiempo ocupa esa actividad.

Los gráficos nos permiten responder preguntas como:

  • ¿Qué actividad ocupa más tiempo?
  • ¿Cuál ocupa menos?
  • ¿Cómo se reparte nuestro tiempo total?

👉 Ahora vas a crear tu propio gráfico con los datos que has recogido.

Gráficas en excell para niños

Actividad. ¿Cómo hacer un gráfico con tus datos?

Una vez introducidos tus datos en la tabla de excell, sigue estos pasos para crear tu gráfico en la hoja de cálculo:

  • 1. Selecciona la tabla

Haz clic y arrastra para seleccionar todos los datos de la tabla.

  •  2. Inserta el gráfico

Busca en el menú la opción: "Insertar → Gráfico".

  •  3. Elige el tipo de gráfico

Selecciona el tipo que quieras:

📊 Gráfico de barras (para comparar).
🟢 Gráfico circular (para ver partes del total).
 

  • 4. Revisa tu gráfico

Comprueba que:

✔ Los datos son correctos
✔ El gráfico tiene título
✔ Se entiende bien
